Animal Crossing: New Horizons is a social simulation game that involves creating a home, interacting with animal villagers, and enjoying everything else the game has to offer such as fishing, visiting other people’s islands, and shopping. A large part of the game involves collecting DIY recipes which allows you to build even more items that aren’t sold in shops. There are various ways to get these DIY recipes such as from your own villagers, message bottles, from other players, balloon presents and more. One category of these DIY recipes are foods and drinks which you are able to make once you have an interactable kitchen item. These food items can become cute decorations to your island and home or even be eaten or drinken to earn energy points which are used to relocate trees or break rocks. This website is created with hand drawn illustrations based on Animal Crossing: New Horizons, and it is a collection of the DIY recipes that are created with fruits in the game. The home page allows the user to create the same food or drink items by interacting with the page. If more information is wanted about each item, there is a page detailing information such as energy received when consuming the item in-game and real recipes that correlate with the items in-game.
